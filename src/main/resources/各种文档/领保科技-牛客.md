1. 爬楼梯算法（动态规划、递归）n层楼梯，一次走1或2步，问：有多少种方式走完这个n层楼梯

   - 动态规划：呈现肥波序列数规律增加，最后一个数就是总的方法数

   - ~~~java
     class Solution{
         //n：台阶总数
         public int test(int n){
             if(n==1 ||n==2){
                 return n;
             }
             int[]dp=new int[n+1];
             //基本情况
             dp[0]=0;
             dp[1]=1;
             dp[2]=2;
             
             //dp方程如何构造：最后一次一定是要么爬1级楼梯，要么爬2级楼梯
             //所以：n级楼梯转化为（n-1）级楼梯的方法+（n-2)级楼梯的方法
             for(int i=3,i<=n;i++){
                 dp[i]=dp[i-1]+dp[i-2];
             }
             return dp[n]
     
          
         }
     }
     ~~~

   - 递归方案

   - ~~~java
     class Soulution{
         public int t(int n){
             if(n<=3){
                 return n;
             }
             return t(n-1)+t(n-2);
         }
     }
     ~~~
